# What
Langflow is a visual builder for creating AI workflows using LangChain concepts without heavy coding.
Think of Langflow as:
“Postman for AI workflows”
or
“Flowchart-based backend for AI systems”
Langflow  - open source alternative of n8n

# Why
Why Langflow Is Important for QA Engineers  
• You see what the AI is doing step by step  
• Easy to debug retrieval problems  
• No black-box prompting  
• Easy to replace models (OpenAI → Local LLM)

I - Instructions

C - Context

E - Examples

P - Person

O - Output

T - Tone
